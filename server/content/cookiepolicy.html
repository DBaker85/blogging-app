This site uses cookies and localstorage to improve the user experience and for statistics

visit

`visit` cookie is used for statistics and to tell if the user has visited the site or not.

siteTheme

`siteTheme` is used to store the user chosen color theme

session

`session` is used to store the logged in session for the admin panel

Collected data

Some data is collected and stored for analytics purposes.

  • Date of unique visit
  • Device used
  • Browser used
  • Geographical data consisting of: Town, City and Country
